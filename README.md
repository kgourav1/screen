# Full Screen Color Display

A clean, simple, and powerful web application for displaying solid colors in full screen with optional time display features. Perfect for photographers, videographers, content creators, and anyone who needs a customizable color screen.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🌟 Features

### Core Functionality

- **Custom Color Selection**: Choose any color using the color picker or enter hex codes directly
- **8 Preset Colors**: Quick access to common colors (Black, White, Red, Green, Blue, Yellow, Magenta, Cyan)
- **True Full Screen**: Native browser full-screen API support for immersive experience
- **Easy Exit Options**:
  - Press `ESC` key
  - Click the hover-revealed close button (×)

### Time Display Features

- **Clock Mode**: Display current time in HH:MM:SS format
- **Timer Mode**: 5-minute countdown timer
- **Stopwatch Mode**: Full-featured stopwatch with:
  - Start/Pause/Resume functionality
  - Reset option
  - Displays hours, minutes, and seconds

### User Experience

- Clean, modern dark-themed interface
- Smooth transitions and animations
- Responsive design (works on desktop, tablet, and mobile)
- Large, readable time display (120px on desktop, 60px on mobile)
- Real-time color preview before going full screen

## 🚀 Getting Started

### Installation

1. Download the HTML file
2. Open it in any modern web browser
3. That's it! No dependencies or server required.

### Usage

1. **Select a Color**:
   - Click on a preset color, OR
   - Use the color picker, OR
   - Type a hex code (e.g., #FF5733)

2. **Choose Time Display** (Optional):
   - Click "None" for just the color screen
   - Click "Clock" to show current time
   - Click "Timer" for a 5-minute countdown
   - Click "Stopwatch" for a stopwatch with controls

3. **Go Full Screen**:
   - Click "Go Full Screen" button
   - Your screen will fill with the selected color

4. **Exit Full Screen**:
   - Press `ESC` key, OR
   - Hover near the top-right and click the × button
